# GirhubStars

A Vue web application to allow the user to assign tags to starred repositories from Github for further searching and filtering.

- The app fecthes repository information from Github server using the Github v3 REST API.
- Tags data is stored in the local storage of the user with the help of the Vuex library.

## Getting started

### Prerequisites

+ [yarn](https://yarnpkg.com/) Package manager
+ [Vue](https://vuejs.org/)
+ [Vue-CLI](https://cli.vuejs.org/)

### Project setup

```shell
yarn install
```

### Run with Vue-CLI:

To easily deploy the application with hot-reload feature activated, use:
```shell
$ vue-cli-service serve
```
or using [yarn](https://yarnpkg.com/)
```shell
$ yarn serve
```
